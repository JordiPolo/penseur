# Penseur

This gem will let you compare two project's gems and tell you how to
make them the same.

## Installation

Install it as:

    $ gem install penseur

## Usage

    $ penseur compare  PROJECT1 PROJECT2

where PROJECT1 and PROJECT2 are directories with your projects
